## Interpretation

The full CSE-CIC-IDS2018 sample independently reproduces the central paper signal: random stratified evaluation is optimistic relative to temporal stress testing.

The result is not a simple model ranking claim:

- HistGradientBoosting and RandomForest are strong on random split but collapse under temporal shift.
- RandomForest remains high on the latest-day holdout, so the temporal collapse is not a universal statement for every stress split.
- Web holdout reduces all three models, with RandomForest showing the largest Web-holdout degradation among the high-random models.
- LogisticRegression is weaker in random split but less catastrophic than tree ensembles on the temporal split.

Use this result as:

> external validation on real CSE-CIC-IDS2018 CICFlowMeter CSVs using a capped stratified sample from all 10 processed CSV files.

Do not present it as:

> a full-prior benchmark over all 16.2M rows or a complete UNSW-NB15 validation.

## Limitations

- The experiment uses a capped stratified sample for tractable local execution.
- Endpoint-pair holdout is not valid on the public processed CSE-CIC-IDS2018 files because endpoint identity fields are absent or incomplete.
- Only seed 42 has been run for this external validation so far.
- The dataset comes from the same CIC/UNB ecosystem as CICIDS-style datasets, so it reduces but does not eliminate external-validity concerns.
